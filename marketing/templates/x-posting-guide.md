# X 投稿ガイド：おくすり味ナビ

---

## 絶対守ること

- 投稿に含める薬×食品の情報は、**必ずアプリのデータと照合**してから投稿する
- 医療的な断定をしない（「絶対に飲める」「これで治る」等はNG）
- フォロワーからの個別の医療相談には「薬剤師にご相談ください」と返す
- 他サイトの文章をコピーしない

## 2. 投稿の種類と頻度

| 種類 | 頻度 | 目的 |
|------|------|------|
| 短文ポスト（保護者向け） | 週2回 | 実用情報で認知拡大 |

## 3. 短文ポストのテンプレート（3種類）

### A.「おすすめ型」（基本。最も使う）

```
【（薬名）の飲ませ方】

おすすめは（食品名）。
（理由を1文で）。

（補足やコツを1文で）。
```

**実例:**

```
【クラリスの飲ませ方】

おすすめはチョコアイス🍫
冷たさ＋甘味で苦味を一番抑えられます。

コツは「混ぜる」より「挟む」。
アイスで薬をサンドして、噛まずに飲み込むと◎
```

```
【ムコダインの飲ませ方】

おすすめはヨーグルト🥄
甘酸っぱい味同士で自然になじみます。

りんごジュースやアイスもOK。
比較的何にでも混ぜやすい薬です。
```

---

### B.「NG警告型」（拡散されやすい）

```
⚠ （薬名）に（食品名）はNGです

（理由を1〜2文で）。

おすすめは（代替食品）。
```

**実例:**

```
⚠ クラリスにスポドリはNGです

酸性でコーティングが溶けて、ものすごく苦くなります。
ヨーグルトも同じ理由でNG。

おすすめはチョコアイスかココア🍫☕
```

```
⚠ ジスロマックにオレンジジュースはNGです

酸性飲料でコーティングが壊れて苦味が激増します。

おすすめはチョコアイスかバニラアイス🍨
冷たくて甘いものが正解です。
```

---

### C.「意外な事実型」（保存されやすい）

```
意外と知られていないこと💊

（知っておくべき事実を2〜3文で）。
```

**実例:**

```
意外と知られていないこと💊

クラリスとムコダインを一緒に処方されたら、
2つの薬は別々に飲ませてください。

ムコダインは酸性なので、混ぜるとクラリスのコーティングが溶けて激苦に。
別々に飲めばどちらも問題なく飲めます。
```

```
意外と知られていないこと💊

粉薬を食べ物に混ぜたら「すぐ飲む」が鉄則です。

時間が経つとコーティングが溶けて苦味が出ます。
作り置きは絶対にNGです。
```

## 4. 画像戦略

テキストだけの投稿よりも画像付きの方がエンゲージメントが高い。

### 効果的な画像

| 画像の種類 | 使いどころ | 作り方 |
|-----------|-----------|--------|
| アプリのスクリーンショット | アプリ紹介投稿、note記事シェア時 | アプリを開いてスマホでキャプチャ |
| おすすめ/NG の比較画像 | NG警告型の投稿 | ◎と✕を並べた簡単な画像（Canva等で作成） |
| 飲み合わせ一覧表 | 意外な事実型、保存促進 | アプリの一覧表画面をキャプチャ |

画像内にもアプリ名を小さく入れておくと、画像が単独で拡散された時にも認知につながる。

## 5. note記事をXでシェアする場合

1. noteに記事を公開する（これがオリジナル）
2. Xで記事リンク付きのポストを投稿する
3. ポストの文面は「記事の結論を1〜2行 + リンク」。長い説明は不要

**シェア投稿の例:**
```
クラリスドライシロップ、何に混ぜたら飲める？

チョコアイスが一番◎
ヨーグルト・スポドリは逆に苦くなるのでNG⚠

公的資料ベースでおすすめとNGをまとめました👇
（noteのURL）
```

## 6. リプライ・DMへの対応方針

| 状況 | 対応 |
|------|------|
| 「参考になりました！」等のお礼 | お礼を返す。余裕があれば関連情報を1つ追加 |
| 「この薬はどうですか？」 | アプリで調べてスクショで回答。または「アプリで確認できます」と誘導 |
| 「子どもが〇〇の症状で…」等の医療相談 | **「薬剤師や医師にご相談ください」と返す。** 個別の医療判断には絶対に踏み込まない |
| 誤情報の指摘 | 真摯に対応。確認して間違いなら即修正 |

## 7. DO / DON'T

### DO
- 1ポスト1メッセージに絞る（詰め込みすぎない）
- 薬の商品名を必ず入れる（検索でヒットさせるため）
- 絵文字は1〜3個で控えめに（💊🍫⚠ など）
- 「保存」されやすい実用情報を優先する
- 出典があることをときどき示す（「公的資料ベースの情報です」等）

### DON'T
- 毎回アプリリンクを貼らない（宣伝アカウントに見える）
- 医療的な断定をしない
- 他の薬局・病院の批判をしない
- ハッシュタグを大量につけない（1〜2個まで）
- 投稿頻度を無理に上げない（質 > 量）

### 使えるハッシュタグ（1投稿に1〜2個）

`#粉薬` `#子育て` `#薬の飲ませ方` `#小児科`

投稿内容に合うものを選ぶ。毎回同じにしない。

## 8. 薬剤師向け投稿（月2回）

市場分析で「薬剤師の口コミ」が主要ユーザー獲得チャネルと判明。
薬剤師が服薬指導の現場でアプリを使い、保護者に紹介する状態を目指す。

### テンプレート

```
【薬剤師の方へ】

保護者への服薬指導で
「何に混ぜたら飲めますか？」と聞かれたとき、

おくすり味ナビを見せながら説明すると
おすすめとNGが一目で伝わります💊

公的資料ベース。複数薬の共通相性も確認できます。
📱（URL）
```

```
【薬剤師の方へ】

クラリスとムコダインが同時処方のとき、
「別々に飲ませてください」の説明、伝わりにくくないですか？

おくすり味ナビの画面を保護者に見せると
「なぜ別々なのか」が視覚的に伝わります。

📱（URL）
```

### 薬剤師向けハッシュタグ

## 9. 季節性カレンダー

検索需要の季節変動に合わせてコンテンツを集中投下する。

| 時期 | テーマ | 重点キーワード |
|------|--------|--------------|
| **12〜2月** | インフルエンザシーズン | タミフル、リレンザ、イナビル |
| **3〜5月** | 花粉症・アレルギー | 抗ヒスタミン薬（アレジオン等） |
| **6〜8月** | 夏風邪・手足口病 | 解熱鎮痛剤（カロナール等） |
| **9〜11月** | マイコプラズマ・秋の感染症 | クラリス、ジスロマック |
| **通年** | 中耳炎・気管支炎 | クラリス、ムコダイン、ワイドシリン |

季節の切り替わり（11月末、2月末等）に該当テーマの投稿を1〜2本集中投下する。
