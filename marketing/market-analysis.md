# おくすり味ナビ — 競合・市場分析レポート

作成日: 2026-03-18

---

## エグゼクティブサマリー

「子供の粉薬を何に混ぜたら飲めるか」を即座に検索できるアプリは、**日本にも世界にも存在しない**。85%の家庭が子供の服薬で困った経験があり、対象児童は約550万人。情報は国立成育医療研究センターのPDFに閉じ込められており、親はGoogle検索と断片的なブログ記事に頼っている。明確なブルーオーシャンである。

## 1. 市場規模（TAM / SAM / SOM）

### TAM（総市場規模）— 子供に薬を飲ませる全ての親

| 指標 | 数値 | 出典 |
|------|------|------|
| 0〜6歳の子供 | 約550万人 | 総務省統計局 |
| 年間出生数 | 68.6万人（2024年） | 厚労省 |
| 小児処方箋（推定） | 年間8,000万〜1.3億枚 | 厚労省需給推計から試算 |
| 薬で困った経験がある家庭 | **85%** | すくコム調査（n=1,786） |
| 粉薬が嫌がられる割合 | **84.2%** | 日本調剤調査 |

### SAM（到達可能市場）— スマホで薬の飲ませ方を調べる親

- 「粉薬 子供 飲ませ方」等のキーワードで検索する親（推定：年間数十万〜百万検索）
- 育児アプリ利用率：約65%の共働き世帯が定期利用
- お薬手帳アプリ利用者：約422万人（2025年予測）

### SOM（短期獲得可能市場）— 初年度の現実的ターゲット

- note/X でのSEO記事経由の流入
- 薬剤師の口コミ
- **初年度目標: 月間1,000〜5,000 UU**（保守的見積）

## 2. 競合分析

### 直接競合: **存在しない**

「粉薬×食品の味の相性」を検索できるアプリ・サービスは日本にも海外にも確認できなかった。

### 間接競合マップ

| 競合 | 種別 | やっていること | おくすり味ナビとの違い |
|------|------|--------------|---------------------|
| **NCCHD飲み合わせPDF** | 公的資料 | 薬×食品の相性一覧表 | PDF。検索不可、スマホで読めない |
| **キャップスクリニック** | 小児科ブログ | 薬別の飲み合わせ記事 | 静的記事。検索・比較不可 |
| **ファーマラボ** | 薬剤師向けサイト | 飲み合わせ一覧表 | 薬剤師向け。親には読みにくい |
| **ミナカラ** | 薬局系メディア | クラリス等の飲ませ方記事 | 記事単位。薬を横断検索不可 |
| **EPARKお薬手帳** | アプリ（400万DL） | 薬×薬の相互作用チェック | 薬×食品の「味」は対象外 |
| **アイン薬局アプリ** | 薬局アプリ | 薬剤師チャット相談 | リアルタイムではない。人依存 |
| **FLAVORx（米国）** | アプリ+サービス | 薬にフレーバーを追加 | 薬局での調剤サービス。家庭のDIYではない |
| **龍角散おくすり飲めたね** | 服薬補助食品 | ゼリー状の飲み込み補助 | 物理的な商品。情報ツールではない |

### ポジショニングマップ

```
          情報の網羅性
              高
              │
    ファーマラボ │  ★ おくすり味ナビ
    (薬剤師向け) │  (親向け・即座に検索)
              │
    ──────────┼──────────── 使いやすさ
    NCCHD PDF  │  キャップスクリニック
    (原典だが  │  (わかりやすいが
     読めない)  │   1記事だけ)
              │
              低
```

**おくすり味ナビの独自ポジション**: 「公的資料の網羅性」×「親が片手で使えるUX」の交差点を唯一占めている。

## 3. Porter's Five Forces（業界構造分析）

### ① 既存競合の脅威: **極めて低い**
- 直接競合アプリはゼロ
- 間接競合（ブログ記事、PDF）はUX面で全く脅威にならない

### ② 新規参入の脅威: **中程度**
- 技術的な参入障壁は低い（NCCHDのPDFは公開データ）
- ただし「薬×食品の相性データをアプリ化する」という発想自体がニッチで、大手が気づきにくい
- **最大のリスク**: NCCHDが自前でデジタル化する可能性。ただし公的機関が優れたモバイルUIを作る可能性は低い

### ③ 代替品の脅威: **中程度**
- 「薬剤師に直接聞く」が最大の代替品（そして現状の主流）
- 服薬補助食品（おくすり飲めたね等）は代替というより補完
- AI（ChatGPT等）に聞く親が増える可能性はあるが、ハルシネーションリスクが医療情報では致命的

### ④ 買い手の交渉力: **高い**
- 親は無料を期待する（育児アプリは無料が標準）
- 直接課金は難しい → 広告・B2B・アフィリエイトが現実的

### ⑤ 供給者の交渉力: **低い**
- データソース（NCCHD）は公開資料で無料
- ホスティング（Cloudflare Pages）は無料
- 運用コストは極めて低い

## 4. SWOT分析

### Strengths（強み）
- **唯一の存在**: この機能を持つアプリが市場にない
- **公的データ**: NCCHDの資料に基づく信頼性
- **即答性**: 薬名で検索 → おすすめ/NG が一目でわかるUI
- **複数薬まとめ機能**: 処方箋に複数の薬がある場合に共通の食品を提示。競合には不可能
- **PWA**: インストール不要。URL共有で広がる
- **運用コストほぼゼロ**: 静的サイト + Cloudflare Pages

### Weaknesses（弱み）
- **認知ゼロ**: 現時点でWeb上に存在感がない
- **データの古さ**: 元データは2004-2007年の論文に基づく。新薬がカバーされていない可能性
- **個人開発**: 開発リソースが限られる
- **飲み合わせ表のモバイルUX**: テーブル形式がスマホで見にくい（要改善）
- **マネタイズ手段が未確立**

### Opportunities（機会）
- **服薬フォローアップ義務化**（2025年薬機法改正）: 薬剤師が服薬指導の質を上げるツールを求めている
- **電子処方箋普及**（薬局の63%が導入済み）: 薬局DXの波に乗れる
- **SEO空白**: 「〇〇 飲ませ方」で上位表示を狙える。現状は個人ブログや病院PDFが占拠
- **龍角散との提携可能性**: おくすり飲めたねをアプリ内で推奨 → 相互送客
- **B2B展開**: 薬局向け服薬指導ツールとしてのSaaS化
- **製薬会社・OTC企業の広告**: ターゲティング精度が高い（子供に薬を飲ませたい親）

### Threats（脅威）
- **少子化**: 対象人口が年々減少（ただし1世代あたりの利用密度は高い）
- **NCCHD自身のデジタル化**: 可能性は低いが、実現すると最強の競合になる
- **大手薬局チェーンの参入**: EPARKやアイン薬局が機能追加する可能性
- **AI相談の普及**: ChatGPT等で「クラリス 何に混ぜる」と聞く親が増える可能性
- **データ更新の停滞**: 元データが更新されなければ、新薬に対応できない

## 5. 検索環境分析

### 主要キーワードの現状

| キーワード | 1位 | アプリの有無 | 機会 |
|-----------|-----|:-----------:|------|
| クラリス 飲ませ方 | ミナカラ（薬局メディア） | なし | 高 |
| 粉薬 子供 飲ませ方 | PMDA（政府機関） | なし | 高 |
| ムコダイン 飲み合わせ | QLife（薬DB） | なし | 高 |
| タミフル 子供 飲ませ方 | 育児ブログ | なし | 高（季節性） |
| 粉薬 何に混ぜる | PMDA | なし | 高 |

**全キーワードでアプリが検索結果に出現しない。** note記事で上位を取れれば、アプリへの導線を独占できる。

### 検索結果の品質
- 上位記事の多くは「一般的な飲ませ方のコツ」に留まり、薬剤別の具体的な相性表を掲載しているサイトは少数
- 情報の更新頻度が低く、数年前の記事がそのまま上位
- **体系的なコンテンツを持つサービスが参入すれば上位表示の可能性は十分ある**

## 7. 戦略提言

### 今やるべきこと（Phase 1）
1. **note記事で検索上位を取る** — 「クラリス 飲ませ方」で1ページ目に入ることが最優先
2. **X で認知を作る** — 週2回の投稿で薬剤師・育児層のフォロワーを獲得
3. **モバイルUXを磨く** — 飲み合わせ表のスマホ対応が最大の技術課題

### 次にやるべきこと（Phase 2）
4. **薬剤師コミュニティへの浸透** — 薬剤師がリアルの服薬指導でアプリを紹介する状態を作る
5. **龍角散への連絡** — おくすり飲めたねとの相互送客提携
6. **データの拡充** — NCCHD以外の情報源（病院独自の資料等）も取り込み、カバー薬品数を増やす

### 守るべきこと
- **「公的資料ベース」の信頼性** — これが最大の差別化。個人の体験談やAI生成情報と混ぜない
- **医療行為の範囲に踏み込まない** — 「情報提供」に徹する。診断・処方判断は絶対にしない
